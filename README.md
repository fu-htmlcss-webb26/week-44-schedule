# Schema, vecka 44
###### HTML & CSS, vecka 3 av 8

## Introduktion

Den här veckan tar vi layouten till nästa nivå. Med CSS Grid får du möjlighet att bygga mer avancerade layouter med rader och kolumner.

Vi arbetar också med formulär – en central del av många webbplatser. Du får lära dig hur formulär byggs med HTML och hur de kan stylas för att bli både användarvänliga och visuellt tilltalande.

## Mål för veckan:

1. Förstå skillnaden mellan Flexbox och CSS Grid.
2. Skapa tvådimensionella layouter med CSS Grid.
3. Välja lämplig layoutteknik utifrån behov.
4. Skapa formulär med HTML.
5. Använda vanliga formulärelement som input, textarea, select och checkbox.
6. Styla formulär med CSS.
